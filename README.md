### About me

I'm a full-stack developer from Sofia who's been writing code since 4th grade — what started as a dojo-themed browser game in pure HTML/CSS/JS turned into a decade of curiosity, late nights, and a lot of shipped projects.

I love building things end-to-end: backend, frontend, database, cloud. Currently sharpening my Azure & cloud-native skills because I think the *deployed* part of "I built this" matters just as much as the code itself.

- 🚀 **Currently shipping:** AVG — a full production platform for a driving school in Pernik (sole engineer, end-to-end)
- 🎓 **Studying:** SoftUni · planning to enroll at university in 2026
- 🏆 **Recent win:** National award at ITA 2025 for **MASHA**, an interactive robot
- 🌱 **Learning more about:** Azure, Kubernetes, distributed systems
- 💬 **Open to:** Junior Full-Stack roles & good conversations about software
